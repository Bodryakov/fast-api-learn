# Описание проекта Fast-API-Learn

## Общее описание
Fast-API-Learn — минималистичное full-stack веб-приложение для обучения основам FastAPI. Backend построен на FastAPI и подключён к Supabase. Frontend реализован на чистом HTML, CSS и VanillaJS без фреймворков. Контент уроков, тестов, задач и изображения хранится в Supabase в формате JSONB. Структура URL строго соответствует формату:
`section-<номер>-<slug>/lesson-<номер>-<slug>`.

При входе пользователь сразу попадает на страницу разделов и уроков, оформленных карточками. Уроки формируются динамически из общего шаблона и данных, полученных из Supabase. Есть отдельная страница 404.

## Особенности
- Редактор Tiptap в админ-панели для создания и редактирования уроков, включая загрузку изображений в Supabase Storage.
- Админка доступна только по адресу `/bod` с фиксированным логином/паролем и cookie-сессией на 30 дней.
- Полная CRUD-логика через REST API FastAPI.
- Материал уроков, тесты и задачи — в JSONB.
- Современный адаптивный дизайн на CSS Grid, Material Design, переключение светлой/тёмной темы.
- Защита от XSS/CSRF на уровне FastAPI и шаблонов.

## База данных
- `sections`: разделы (номер, заголовок, slug, порядок, метаданные).
- `lessons`: уроки (привязка к разделам, номер, slug, статус, JSONB-контент).
- Уникальные ограничения на номер и slug.
- Supabase Storage: бакет `lesson-images`.

## Как будет создаваться проект
1. Создать SQL-скрипт таблиц Supabase.
2. Создать SQL-скрипт заполнения тестовыми данными (2 раздела × 2 урока).
3. Создать виртуальное окружение.
4. Создать `requirements.txt` для деплоя на Koyeb.
5. Установить зависимости.
6. Создать `Procfile` с командой запуска.
7. Сгенерировать код приложения согласно структуре.
